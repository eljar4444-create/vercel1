1. **Data fetching** (copy and adapt from TopMastersSection.tsx):
   ```typescript

     where: { user: { isBanned: false }, status: 'PUBLISHED', is_verified: true },
     take: 9,  // enough for 3 rows of 3
     include: {
       reviews: { select: { rating: true } },
       category: { select: { name: true } },
       user: { select: { image: true } },
       services: { select: { title: true, price: true, duration_min: true }, take: 2 },
     },
     orderBy: { created_at: 'desc' },
   });
   ```

2. **Section wrapper**: `<section className="py-12 md:py-16 bg-booking-bg">`

4. **Gallery grid**:
   - Desktop: `grid grid-cols-2 lg:grid-cols-3 gap-6`
   - Mobile: `flex flex-col gap-6`

5. **Render logic**: Map masters. After the first 6 cards (row 2 on desktop), insert `<JealousyCard />` as a grid item spanning full width (`col-span-2 lg:col-span-3`). Then render remaining masters.
